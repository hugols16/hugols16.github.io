Webcam Tracking Software

Using OpenCV, I implemented a Python module to track objects based on their HSV color range. This included:

  • Frame Segmentation and Reduction - Breaking down the video frame by frame and reducing its size by 16 to optimize its pixel reading.
  • Mask Implementation - Masking each frame with a HSV range using NumPy.
  • Object Identification - Finding the biggest positive area and marking it with a red rectangle.
  • Movement Adaptation - Creating a function to simulate the responsed movement from the webcam, for smoother visuals
